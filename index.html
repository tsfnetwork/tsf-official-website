TSF SPEC .

TSF Coin Spec

  • POW Algorithm: Dagger Hashimoto (Ethash)

  • PoW Block Reward: Fixed 2 TSF

  • Block Time: 45 seconds

  • Difficulty: Retargets at every block

  • RPC port: 4949

  • P2P port: 59997

  • WebSocket (ws) port: 4950

  • Max emission of coins: Infinite

  • Starting Testnet: --testnet --mine

How to mine TSF

  • LolMiner:

    lolMiner --algo ETHASH --pool tsfpool.xyz:9449 --user <address> --worker <worker>.

  • PhoenixMiner

    PhoenixMiner -pool tsfpool.xyz:9449 -wal <address> -pass x -worker <worker> -coin tsf.

  • TeamRedMiner

    teamredminer -a ethash -o stratum+tcp://tsfpool.xyz:9449 -u <address> -p x --eth_worker <worker>.

  • EthMiner

    ethminer.exe -F http://tsfpool.xyz:8888/ <address> / <worker> -G.
